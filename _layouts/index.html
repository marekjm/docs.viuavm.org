Manuals

Manuals are short articles explaining how to use specific features of the machine, intended to be reminders (a bit like man pages on *nix-like system) for people who already know what they are looking for. The reader is assumed to have prior C++ and Viua VM programming experience.

Tutorials

Tutorials are articles of varying length, explaining how to program Viua VM, and related projects (e.g. how to write a C++ extension module). These articles are written as easy to follow step-by-step guides, explaining majority of the things as they are encountered in text. The reader is expected to have prior programming experience, and be reasonably proficient in the craft.

References

References also contain VM instruction set reference, and bytecode format description.

References are lists of features, with short examples of usage (in the spirit of cppreference.com). They are intended either for people who just need to refresh their memory (what arguments does a function take, in which module was a function implemented, etc.), or people who are interested in feature lists.
