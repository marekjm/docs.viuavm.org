Syntax

{% highlight lua %} izero r0 {% endhighlight %}

Semantics

Stores a 64 bit signed integer with the value of 0 in a register specified by "r0".

Examples

{% highlight lua %} .function: example/0 -- store "0" in local register 1 izero %1 local return .end {% endhighlight %}

Notes

This instruction is equivallent to "istore r0 0".
