Syntax

{% highlight lua %} {{ page.canonical_form }} bits destination:r-op width:r-op {% endhighlight %}

Examples

Bit literals can be written as binary, octal, or hexadecimal numbers. Alternatively, bits can be initialised by setting their width in which case they will be filled with zeroes.

{% highlight lua %} .function: example/0 bits %1 local 0b100111010 bits %2 local 0o755 bits %3 local 0xdeadbeef istore %4 local 8 bits %4 local %4 local return .end {% endhighlight %}
