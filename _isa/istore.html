--- title: istore layout: isa_opcode isa_group: arithmetic canonical_form: 'istore {:r-operand} {:integer}' summary: > Puts a 64 bit signed integer into a register. ---

Syntax

{% highlight lua %} (0) istore r0 i0 {% endhighlight %}

Semantics

Stores a 64 bit signed integer in a register specified by "r0".

Examples

{% highlight lua %} .function: example/0 -- store "42" in local register 1 istore %1 local 42 return .end {% endhighlight %}
