--- title: text layout: isa_opcode isa_group: text canonical_form: 'text Destination:r-op "utf8-content"' summary: > Puts a text into a register. ---

Syntax

{% highlight lua %} (0) {{ page.canonical_form }} (1) text Destination Source:r-op {% endhighlight %}

Semantics

Form (0) stores an UTF-8 encoded text value in a register. Form (1) converts any value to text.

Exceptions

Form (0) triggers an exception when the text to store is not a valid UTF-8 sequence.

Examples

{% highlight lua %} .function: main/0 text %1 local "Hello World!" text %2 local "Adieu World!" texteq %3 local %1 local %2 local print %3 local istore %4 local 42 text %5 local %4 local -- convert integer to text print %5 local izero %0 local return .end {% endhighlight %}
