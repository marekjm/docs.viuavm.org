Below you can find manuals needed when writing software for Viua VM written as articles of varying length, accompanied by code samples in Viua assembly and, occasionally, other languages. Manuals are thought as resources touching upon more advanced features of the VM, such as: concurrency, message passing, dynamic dispatch, and watchdog process; they also elaborate on VM internals and contain in-depth information about "how-and-why's" of discussed features.

Intended audience

A reasonable proficiency in programming is assumed. For some articles, prior experience with Viua VM, C++ or both is expected. Experience with command line environment is expected.
